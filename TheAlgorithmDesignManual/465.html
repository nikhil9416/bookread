14.5 GENERATING SUBSETS 455 Gray codes were ﬁrst developed [Gra53] to transmit digital information in a robust manner over an analog channel. By assigning the code words in Gray code order, the ith word diﬀers only slightly from the (i + 1)st, so minor ﬂuctuations in analog signal strength corrupts only a few bits. Gray codes have a particularly nice correspondence to Hamiltonian cycles on the hypercube. Savage [Sav97] gives an excellent survey of Gray codes (minimum change orderings) for a large class of combinatorial objects, including subsets. The popular puzzle Spinout, manufactured by ThinkFun (formerly Binary Arts Cor- poration), is solved using ideas from Gray codes.
