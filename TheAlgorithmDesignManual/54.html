The nested inner loop goes around n − i − 1 times, where i is the index of the outer loop. The exact number of times the if statement is executed is given by: S(n)= n−1  i=0 n−1  j=i+1 1= n−1  i=0 n − i − 1 What this sum is doing is adding up the integers in decreasing order starting from n − 1, i.e. S(n)=(n − 1) + (n − 2) + (n − 3) + ... +2+1 How can we reason about such a formula? We must solve the summation formula using the techniques of Section 1.3.5 (page 17) to get an exact value. But, with the Big Oh we are only interested in the order of the expression. One way to think about it is that we are adding up n − 1 terms, whose average value is about n/2. This yields S(n) ≈ n(n − 1)/2.
