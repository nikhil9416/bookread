508 15. GRAPH PROBLEMS: POLYNOMIAL-TIME Minimum-cut methods have found many applications in computer vision, including image segmentation. Boykov and Kolmogorov [BK04] report on an experimental evalua- tion of minimum-cut algorithms in this context. A nonﬂow-based algorithm for edge k-connectivity in O(kn 2 ) is due to Matula [Mat87]. Faster k-connectivity algorithms are known for certain small values of k. All three- connected components of a graph can be generated in linear time [HT73a], while O(n 2 ) suﬃces to test 4-connectivity [KR91].
