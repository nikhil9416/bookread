The actual deletion operation is simple, once ruling out the case that the to-be-deleted element does not exist. Special care must be taken to reset the pointer to the head of the list (l) when the ﬁrst element is deleted: delete_list(list **l, item_type x) { list *p; /* item pointer */ list *pred; /* predecessor pointer */ list *search_list(), *predecessor_list(); p = search_list(*l,x); if (p != NULL) { pred = predecessor_list(*l,x); if (pred == NULL) /* splice out out list */ *l = p->next; else pred->next = p->next; free(p); /* free memory used by node */ } } C language requires explicit deallocation of memory, so we must free the deleted node after we are ﬁnished with it to return the memory to the system. 3.1.3 Comparison The relative advantages of linked lists over static arrays include: • Overﬂow on linked structures can never occur unless the memory is actually full.
