The new edgenode is inserted at the begin- ning of the appropriate adjacency list, since order doesn’t matter. We parameterize our insertion with the directed Boolean ﬂag, to identify whether we need to insert two copies of each edge or only one. Note the use of recursion to solve this problem: insert_edge(graph *g, int x, int y, bool directed) { edgenode *p; /* temporary pointer */ p = malloc(sizeof(edgenode)); /* allocate edgenode storage */ p->weight = NULL; p->y = y; p->next = g->edges[x]; g->edges[x] = p; /* insert at head of list */ g->degree[x] ++; if (directed == FALSE) insert_edge(g,y,x,TRUE); else g->nedges ++; } Printing the associated graph is just a matter of two nested loops, one through vertices, the other through adjacent edges:
