These kind of tests can be done with simple BFS/DFS. The minimum spanning tree of a graph is unique if all m edge weights in the graph are distinct. Otherwise the order in which Prim’s/Kruskal’s algorithm breaks ties determines which minimum spanning tree is returned. There are two important variants of a minimum spanning tree that are not solvable with these techniques. • Steiner Tree – Suppose we want to wire a bunch of houses together, but have the freedom to add extra intermediate vertices to serve as a shared junction. This problem is known as a minimum Steiner tree, and is discussed in the catalog in Section 16.10. • Low-degree Spanning Tree – Alternately, what if we want to ﬁnd the mini- mum spanning tree where the highest degree node in the tree is small? The lowest max-degree tree possible would be a simple path, and have n − 2 nodes of degree 2 with two endpoints of degree 1.
