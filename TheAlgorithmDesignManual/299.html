For edit distance, this might mean printing out the name of the operation or character involved, as determined by the needs of the application. insert_out(char *t, int j) { printf("I"); } delete_out(char *s, int i) { printf("D"); } match_out(char *s, char *t, int i, int j) { if (s[i]==t[j]) printf("M"); else printf("S"); } All of these functions are quite simple for edit distance computation. However, we must confess it is diﬃcult to get the boundary conditions and index manipula- tions correctly.
