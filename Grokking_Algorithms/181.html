162 Chapter 9 I Dynamic programming You have three items that you can put into the knapsack. What items should you steal so that you steal the maximum money’s worth of goods? The simple solution e simplest algorithm is this: you try every possible set of goods and ﬁnd the set that gives you the most value. is works, but it’s really slow. For 3 items, you have to calculate 8 possible sets. For 4 items, you have to calculate 16 sets. With every item you add, the number of sets you have to calculate doubles! is algorithm takes O(2^n) time, which is very, very slow.
