121 Terminology To calculate the shortest path in an unweighted graph, use breadth-ﬁrst search. To calculate the shortest path in a weighted graph, use Dijkstra’s algorithm. Graphs can also have cycles. A cycle looks like this. It means you can start at a node, travel around, and end up at the same node. Suppose you’re trying to ﬁnd the shortest path in this graph that has a cycle. Would it make sense to follow the cycle? Well, you can use the path that avoids the cycle. Or you can follow the cycle.
