So it needs a mapping from page URL to page data. When you visit a page on Facebook, it ﬁrst checks whether the page is stored in the hash. Here it is in code: cache = {} def get_page(url): if cache.get(url): return cache[url] Returns cached data else: data = get_data_from_server(url) cache[url] = data Saves this data in your cache ﬁrst return data Here, you make the server do work only if the URL isn’t in the cache. Before you return the data, though, you save it in the cache. e next time someone requests this URL, you can send the data from the cache instead of making the server do the work.
