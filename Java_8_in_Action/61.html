Functional interface Remember the interface Predicate<T> you created in chapter 2 so you could parameterize the behavior of the filter method? It’s a functional interface! Why? Because Predicate specifies only one abstract method: public interface Predicate<T>{ boolean test (T t); } In a nutshell, a functional interface is an interface that specifies exactly one abstract method.
