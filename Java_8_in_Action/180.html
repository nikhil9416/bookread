Note that in the previous example, there are no guarantees about what type of Set is returned. But by using toCollection, you can have more control. Partitioning Partitioning is a special case of grouping: having a predicate (a function returning a boolean), called a partitioning function, as a classification function. The fact that the partitioning function returns a boolean means the resulting grouping Map will have a Boolean as a key type and therefore there can be at most two different groups—one for true and one for false.
