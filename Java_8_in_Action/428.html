Scala traits vs. Java 8 interfaces Scala has another useful feature for abstraction called traits. They’re Scala’s replacement for Java’s interfaces. A trait can define both abstract methods and methods with a default implementation. Traits can also be multiple inherited just like interfaces in Java, so you can see them as similar to Java 8’s interfaces that support default methods.
