This also demonstrates that using the right data structure and then making it work in parallel guarantees the best performance. Nevertheless, keep in mind that parallelization doesn’t come for free. The parallelization process itself requires you to recursively partition the stream, assign the reduction operation of each substream to a different thread, and then combine the results of these operations in a single value.
