If the ply depth is d, the number of game states checked for tic-tac-toe is ∑ i =1 d b ! b − i ! where b! is the factorial of b. To give an example of the scale involved, Minimax evaluates 187,300 states when b = 10 and d = 6. During the recursive invocation within Minimax, the score(state, player) evalua‐ tion function must be consistently applied using the original player for whom a move is being calculated. This coordinates the minimum and maximum recursive evaluations. Solution The helper class MoveEvaluation pairs together an IMove and an int evaluation to be associated with that move.
