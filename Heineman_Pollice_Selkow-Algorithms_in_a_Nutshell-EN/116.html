Use Sequential Search on small lists. A popular technique is to produce a value based on each character from the original string: hashCode(s) = s[0]*31 (len–1) + s[1]*31 (len–2) + ... + s[len – 1] where s[i] is the i th character (as an ASCII value between 0 and 255) and len is the length of the string s.
