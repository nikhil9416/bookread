Breadth-First Search generates copies of each board state, because moves are applied directly to the boards and not undone. Example 7-7 shows the implementation. Example 7-7. INodeSet closed = StateStorageFactory.create (StateStorageFactory.HASH); while (!open.isEmpty()) { INode n = open.remove(); closed.insert (n); // All successor moves translate into appended OPEN states.
