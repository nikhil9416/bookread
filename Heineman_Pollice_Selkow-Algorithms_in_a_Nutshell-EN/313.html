Input/Output The input is a set of n points P in d-dimensional space and a d-dimensional hyper‐ cube that specifies the desired range query. The range queries are aligned properly with the axes in the d-dimensional data set because they are specified by d individ‐ ual ranges, for each dimension of the input set. For d = 2, the range query has both a range over x coordinates and a range over y coordinates. Range Query generates the full set of points enclosed by the range query. The points do not appear in any specific order. Context k-d trees become unwieldy for a large number of dimensions, so this algorithm and overall approach should be restricted to small dimensional data.
