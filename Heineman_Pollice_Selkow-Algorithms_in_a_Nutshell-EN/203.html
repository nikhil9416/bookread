A search tree represents the set of intermediate board states as a path-finding algorithm progresses. The computed structure is a tree because the algorithm ensures it does not visit a board state twice. The algo‐ rithm decides the order of board states to visit as it attempts to reach the goal. We’ll explore search trees using an 8-puzzle, which is played on a 3×3 grid contain‐ ing eight square tiles numbered 1 to 8 and an empty space that contains no tile. A tile adjacent (either horizontally or vertically) to the empty space can be moved by sliding it into the empty space. The aim is to start from a shuffled initial state and move tiles to achieve the goal state.
