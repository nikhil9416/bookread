The time-complexity of the construction of a suffix tree using the above algorithm is O(n 2 ) where n is the length of the input string because there are n distinct suffixes. The longest has length n, the second longest has length n – 1, and so on. Note: • There are O(n) algorithms for constructing suffix trees. • To improve the complexity, we can use indices instead of string for branches.
