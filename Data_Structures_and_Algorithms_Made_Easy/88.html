Space Complexity: O(1), for creating one temporary variable. 3.7 Doubly Linked Lists The advantage of a doubly linked list (also called two – way linked list ) is that given a node in the list, we can navigate in both directions. A node in a singly linked list cannot be removed unless we have the pointer to its predecessor. But in a doubly linked list, we can delete a node even if we don’t have the previous node’s address (since each node has a left pointer pointing to the previous node and canmove backward). The primary disadvantages of doubly linked lists are: • Each node requires an extra pointer, requiring more space. • The insertion or deletion of a node takes a bit longer (more pointer operations).
