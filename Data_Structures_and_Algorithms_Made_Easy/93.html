Doubly Linked List Deletion Similar to singly linked list deletion, here we have three cases: • Deleting the first node • Deleting the last node • Deleting an intermediate node Deleting the First Node in Doubly Linked List In this case, the first node (current head node) is removed fromthe list. It can be done in two steps: • Create a temporary node which will point to the same node as that of head. • Now, move the head nodes pointer to the next node and change the heads left pointer to NULL. Then, dispose of the temporary node.
