Performance Let n be the number of elements in the stack. The complexities for operations with this representation can be given as: Space Complexity (for n push operations) O(n) Time Complexity of CreateStack() O(1) Time Complexity of PushQ O(1) (Average) Time Complexity of PopQ O(1) Time Complexity of Top() O(1) Time Complexity of IsEmpryStackf) O(1)) Time Complexity of IsFullStackf) O(1) Time Complexity of DeleteStackQ O(1) Note: Too many doublings may cause memory overflow exception. Linked List Implementation The other way of implementing stacks is by using Linked lists. Push operation is implemented by inserting element at the beginning of the list. Pop operation is implemented by deleting the node from the beginning (the header/top node).
