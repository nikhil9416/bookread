Problem-18 Find the Number Occurring an Odd Number of Times: Given an array of positive integers, all numbers occur an even number of times except one number which occurs an odd number of times. Find the number in O(n) time & constant space. Example : I/P = [1,2,3,2,3,1,3] O/P = 3 Solution: Do a bitwise XOR of all the elements. We get the number which has odd occurrences. This is because, A XOR A = 0. Time Complexity: O(n). Space Complexity: O(1).
