21.2.5 Bitwise Right Shift The bitwise right shift moves all bits in the number to the right. Note the use of ? for the fill bits. Where the left shift filled the vacated positions with 0, a right shift will do the same only when the value is unsigned. If the value is signed then a right shift will fill the vacated bit positions with the sign bit or 0, whichever one is implementation-defined. So the best option is to never right shift signed values. 21.2.6 Bitwise Complement The bitwise complement inverts the bits in a single binary number. 21.2.7 Checking Whether K-th Bit is Set or Not Let us assume that the given number is n.
