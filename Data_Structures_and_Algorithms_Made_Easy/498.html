Properties of Bipartite Graphs • A graph is called bipartite if and only if the given graph does not have an odd length cycle. • A complete bipartite graph K m,n is a bipartite graph that has each vertex from one set adjacent to each vertex from another set. • A subset of edges M ⊂ E is a matching if no two edges have a common vertex. As an example, matching sets of edges are represented with dotted lines. A matching M is called maximum if it has the largest number of possible edges. In the graphs, the dotted edges represent the alternative matching for the given graph. • A matching M is perfect if it matches all vertices. We must have V 1 =V 2 in order to have perfect matching. • An alternating path is a path whose edges alternate between matched and unmatched edges. If we find an alternating path, then we can improve the matching. This is because an alternating path consists of matched and unmatched edges. The number of unmatched edges exceeds the number of matched edges by one.
