Solution: Yes, using hash table. Since our objective is to find two indexes of the array whose sum is K. Let us say those indexes are X and Y . That means, A [X]+ A [Y ] =K. What we need is, for each element of the input array A [X], check whether K–A [X] also exists in the input array. Now, let us simplify that searching with hash table. Algorithm: • For each element of the input array, insert it into the hash table. Let us say the current element is A [X]. • Before proceeding to the next element we check whether K – A [X] also exists in the hash table or not. • Ther existence of such number indicates that we are able to find the indexes. • Otherwise proceed to the next input element.
