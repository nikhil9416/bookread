From the diagram we can infer the following properties: • The number of nodes n in a full binary tree is 2 h+1 – 1. Since, there are h levels we need to add all nodes at each level [2 0 +2 1 + 2 2 + ··· + 2 h =2 h+1 – 1]. • The number of nodes n in a complete binary tree is between 2 h (minimum) and 2 h+1 – 1 (maximum). For more information on this, refer to Priority Queues chapter. • The number of leaf nodes in a full binary tree is 2 h . • The number of NULL links (wasted pointers) in a complete binary tree of n nodes is n + 1.
