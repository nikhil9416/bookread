Time Complexity: O(L), where L is the length of the string to be inserted. Searching in Ternary Search Tree If after inserting the words we want to search for them, then we have to follow the same rules as that of binary search. The only difference is, in case of match we should check for the remaining characters (in eq subtree) instead of return. Also, like BSTs we will see both recursive and non- recursive versions of the search method.
