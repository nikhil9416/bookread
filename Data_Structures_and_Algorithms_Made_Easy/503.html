Solution: A vertex i is a sink if and only if M[i,j ] = 0 for all j and M[j, i ] = 1 for all j ≠ i . For any pair of vertices i and j : Algorithm: • Start at i= 1,j= 1 • If M[i,j ]=0→ i wins, j++ • If M[i ,j ]=1→ j wins, i ++ • Proceed with this process until j=n or i=n +1 • If i == n + 1, the graph does not contain a sink • Otherwise, check row i – it should be all zeros; and check column i – it should be all but M[i, i ] ones; – if so, t is a sink. Time Complexity: O(V), because at most 2|V| cells in the matrix are examined. Problem-52 What is the worst – case memory usage of DFS?
