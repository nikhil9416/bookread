Time Complexity: O(nlogn), for sorting. Space Complexity: O(1). Problem-32 Given an array of n elements. Find three elements in the array such that their sum is equal to given element K? Brute Force Solution: The default solution to this is, for each pair of input elements check whether there is any element whose sum is K. This we can solve just by using three simple for loops.
