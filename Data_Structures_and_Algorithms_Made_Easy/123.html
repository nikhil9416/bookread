3.12 Linked Lists: Problems & Solutions Problem-1 Implement Stack using Linked List. Solution: Refer to Stacks chapter. Problem-2 Find n th node from the end of a Linked List. Solution: Brute-Force Method: Start with the first node and count the number of nodes present after that node. If the number of nodes is < n – 1 then return saying “fewer number of nodes in the list”. If the number of nodes is > n – 1 then go to next node. Continue this until the numbers of nodes after current node are n – 1. Time Complexity: O(n 2 ), for scanning the remaining list (from current node) for each node. Space Complexity: O(1).
