Problem-38 Given a matrix with n rows and m columns (n × m). In each cell there are a number of apples. We start fromthe upper-left corner of the matrix. We can go down or right one cell. Finally, we need to arrive at the bottom-right corner. Find the maximum number of apples that we can collect. When we pass through a cell, we collect all the apples left there. Solution: Let us assume that the given matrix is A [n][m]. The first thing that must be observed is that there are at most 2 ways we can come to a cell - fromthe left (if it’s not situated on the first column) and from the top (if it’s not situated on the most upper row).
