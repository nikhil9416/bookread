N(h)= N(h – 1) + N(h – 1) + 1 = 2N(h – 1) + 1 The above expression defines the case of full binary tree. Solving the recurrence we get: N(h) = O(2 h ) ⇒ h = logn ≈ O(logn) ∴ In both the cases, AVL tree property is ensuring that the height of an AVL tree with n nodes is O(logn).
