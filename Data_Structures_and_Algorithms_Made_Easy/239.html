Binary Trees: Problems & Solutions Problem-1 Give an algorithm for finding maximum element in binary tree. Solution: One simple way of solving this problem is: find the maximum element in left subtree, find the maximum element in right sub tree, compare them with root data and select the one which is giving the maximum value. This approach can be easily implemented with recursion.
