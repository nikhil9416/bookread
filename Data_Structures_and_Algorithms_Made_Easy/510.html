• Stable: Maintains relative order of input data if the keys are same • In-place: It requires only a constant amount O(1) of additional memory space • Online: Insertion sort can sort the list as it receives it Algorithm Every repetition of insertion sort removes an element from the input data, and inserts it into the correct position in the already-sorted list until no input elements remain. Sorting is typically done in-place. The resulting array after k iterations has the property where the first k + 1 entries are sorted.
