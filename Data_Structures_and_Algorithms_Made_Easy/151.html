Problem-44 Given a linked list with even and odd numbers, create an algorithm for making changes to the list in such a way that all even numbers appear at the beginning. Solution: To solve this problem, we can use the splitting logic. While traversing the list, split the linked list into two: one contains all even nodes and the other contains all odd nodes. Now, to get the final list, we can simply append the odd node linked list after the even node linked list. To split the linked list, traverse the original linked list and move all odd nodes to a separate linked list of all odd nodes. At the end of the loop, the original list will have all the evennodes and the odd node list will have all the odd nodes. To keep the orderingof all nodes the same, we must insert all the odd nodes at the end of the odd node list. Time Complexity: O(n). Space Complexity: O(1). Problem-45 In a linked list with n nodes, the time taken to insert an element after an element pointed by some pointer is (A) O(1) (B) O(logn) (C) O(n) (D) O(nlogn) Solution: A.
