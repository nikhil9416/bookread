In a Preorder sequence, leftmost element denotes the root of the tree. So we know ‘A’ is the root for given sequences. By searching ‘A’ in Inorder sequence we can find out all elements on the left side of ‘A’, which come under the left subtree, and elements on the right side of ‘A’, which come under the right subtree. So we get the structure as seen below. We recursively follow the above steps and get the following tree. Algorithm: BuildTree() 1 Select an element from Preorder. Increment a Preorder index variable (preOrderIndex in code below) to pick next element in next recursive call. 2 Create a new tree node (newNode) fromheap with the data as selected element. 3 Find the selected element’s index in Inorder. Let the index be inOrderIndex. 4 Call BuildBinaryTree for elements before inOrderIndex and make the built tree as left subtree of newNode. 5 Call BuildBinaryTree for elements after inOrderIndex and make the built tree as right subtree of newNode. 6 return newNode.
