Solution: Refer Searching chapter for discussion on Binary Search. Analysis: Let us assume that input size is n and T(n) defines the solution to the given problem. The elements are in sorted order. In binary search we take the middle element and check whether the element to be searched is equal to that element or not. If it is equal then we return that element. If the element to be searched is greater than the middle element then we consider the right sub- array for finding the element and discard the left sub-array. Similarly, if the element to be searched is less than the middle element then we consider the left sub-array for finding the element and discard the right sub-array.
