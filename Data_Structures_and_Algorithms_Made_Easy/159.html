Time Complexity: O(n). Space Complexity: O(1). Problem-53 Given two linked lists, each list node with one integer digit, add these two linked lists. The result should be stored in the third linked list. Also note that the head node contains the most significant digit of the number. Solution: Since the integer addition starts from the least significant digit, we first need to visit the last node of both lists and add themup, create a new node to store the result, take care of the carry if any, and link the resulting node to the node which will be added to the second least significant node and continue. First of all, we need to take into account the difference in the number of digits in the two numbers. So before starting recursion, we need to do some calculation and move the longer list pointer to the appropriate place so that we need the last node of both lists at the same time. The other thing we need to take care of is carry. If two digits add up to more than10, we need to forward the carry to the next node and add it. If the most significant digit addition results in a carry, we need to create an extra node to store the carry.
