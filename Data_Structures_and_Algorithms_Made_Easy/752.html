Space Complexity: O(n). Problem-13 Catalan Numbers: How many binary search trees are there with n vertices? Solution: Binary Search Tree (BST) is a tree where the left subtree elements are less than the root element, and the right subtree elements are greater than the root element. This property should be satisfied at every node in the tree. The number of BSTs with n nodes is called Catalan Number and is denoted by C n . For example, there are 2 BSTs with 2 nodes (2 choices for the root) and 5 BSTs with 3 nodes.
