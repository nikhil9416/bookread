Due to this, the time complexity of UNION operation is O(1). A FIND(X) on element X is performed by returning the root of the tree containing X. The time to perform this operation is proportional to the depth of the node representing X. Using this method, it is possible to create a tree of depth n- 1 (Skew Trees). The worst-case running time of a FIND is O(n) and m consecutive FIND operations take O(mn) time in the worst case.
