Problem-5 Given an array of n numbers. Give an algorithm for finding the element which appears the maximum number of times in the array? Brute Force Solution: One simple solution to this is, for each input element check whether there is any element with the same value, and for each such occurrence, increment the counter. Each time, check the current counter with the max counter and update it if this value is greater than max counter. This we can solve just by using two simple for loops.
