Symbol tables can be implemented in many ways and some of them are listed below. Unordered Array Implementation With this method, just maintaining an array is enough. It needs O(n) time for searching, insertion and deletion in the worst case. Ordered [Sorted] Array Implementation In this we maintain a sorted array of keys and values. • Store in sorted order by key • keys[i] = i th largest key • values[i] = value associated with i th largest key
