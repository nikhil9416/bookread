Problem-6 For Problem-5 is it possible to improve the space complexity? Solution: Yes. The main problemof the previous approach is, for each push operation we are pushing the element on to min stack also (either the new element or existing minimum element). That means, we are pushing the duplicate minimum elements on to the stack. Now, let us change the algorithm to improve the space complexity. We still have the min stack, but we only pop fromit when the value we pop fromthe main stack is equal to the one on the min stack. We only push to the min stack when the value being pushed onto the main stack is less than or equal to the current min value. In this modified algorithm also, if we want to get the minimum then we just need to return the top element from the min stack. For example, taking the original version and pushing 1 again, we’d get: Main stack Min stack 1 → top 5 1 4 1 → top 6 1 2 2 Popping from the above pops from both stacks because 1 == 1, leaving: Main stack Min stack 5 → top 1 4 6 1 → top 2 2 Popping again only pops fromthe main stack, because 5 > 1: Main stack Min stack 1 → top 4
