Once the tree is built, each leaf node corresponds to a letter with a code. To determine the code for a particular node, traverse fromthe root to the leaf node. For each move to the left, append a 0 to the code, and for each move to the right, append a 1. All we need to do for this calculation is see how many bits are originally used to store the data and subtract from that the number of bits that are used to store the data using the Huffman code. In the above example, since we have six characters, let’s assume each character is stored with a three bit code. Since there are 133 such characters (multiply total frequencies by 3), the total number of bits used is 3 * 133 = 399.
