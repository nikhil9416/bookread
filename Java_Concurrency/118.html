Using FutureTask to preload data that is needed later. Preloader creates a FutureTask that describes the task of loading product information from a database and a thread in which the computation will be per- formed. It provides a start method to start the thread, since it is inadvisable to start a thread from a constructor or static initializer. When the program later needs the ProductInfo, it can call get, which returns the loaded data if it is ready, or waits for the load to complete if not.
