Binding a long-running task to a visual component. Listing 9.5 illustrates the obvious way to do this, which is starting to get com- plicated; we’re now up to three layers of inner classes. The action listener ﬁrst dims the button and sets a label indicating that a computation is in progress, then submits a task to the background executor.
