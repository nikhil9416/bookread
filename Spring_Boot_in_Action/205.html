188 appendix B Spring Boot starters Spring Boot starter dependencies greatly simplify the dependencies section of your project’s build specification by aggregating commonly used dependencies under more coarse-grained dependencies. Your build will transitively resolve the depen- dencies that are declared in the starter dependency. Not only do starter dependencies keep the dependencies section of the build smaller, they are typically organized by the type of functionality they bring to an application. For example, rather than specify specific libraries required for valida- tion (such as Hibernate Validator and Tomcat’s embedded expression language), you can simply add the spring-boot-starter-validation starter as a dependency. Table B.1 lists all of Spring Boot’s starter dependencies along with the depen- dencies that they transitively declare.
