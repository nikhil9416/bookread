20 CHAPTER 1 Bootstarting Spring USING THE INITIALIZR FROM THE SPRING BOOT CLI As you saw earlier, the Spring Boot CLI is a great way to develop Spring applications by just writing code. However, the Spring Boot CLI also has a few commands that can help you use the Initializr to kick-start development on a more traditional Java project. The Spring Boot CLI includes an init command that acts as a client interface to the Initializr. The simplest use of the init command is to create a baseline Spring Boot project: $ spring init After contacting the Initializr web application, the init command will conclude by downloading a demo.zip file.
