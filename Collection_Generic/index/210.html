Adding an Element to a Queue The exception-throwing variant of this operation is the add method inherited from Collection. Although add does return a boolean signi- fying its success in inserting an element, that value can’t be used to report that a boun- ded queue is full; the contract for add specifies that it may return false only if the collection refused the element because it was already present—otherwise, it must throw an exception. The value-returning variant is offer: boolean offer (E e) // insert the given element if possible The value returned by offer indicates whether the element was successfully inserted or not.
