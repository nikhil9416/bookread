Comparator<String> sizeOrder = new Comparator<String>() { public int compare(String s1, String s2) { return s1.length() < s2.length() ? -1 : s1.length() > s2.length() ? 1 : s1.compareTo(s2) ; } }; Here is an example: assert "two".compareTo("three") > 0; assert sizeOrder.compare("two","three") < 0; In the natural alphabetic ordering, "two" is greater than "three", whereas in the size ordering it is smaller. The Java libraries always provide a choice between Comparable and Comparator. For every generic method with a type variable bounded by Comparable, there is another generic method with an additional argument of type Comparator.
