the adapter pattern you are here 4 241 Adaptee Client Adapter request() translatedRequest() The Adapter Pattern explained The Client is implemented against the target interface. The Adapter implements the target interface and holds an instance of the Adaptee. target interface adaptee interface Now that we have an idea of what an Adapter is, let’s step back and look at all the pieces again. The client makes a request to the adapter by calling a method on it using the target interface. The adapter translates the request into one or more calls on the adaptee using the adaptee interface. The client receives the results of the call and never knows there is an adapter doing the translation.
