34 Chapter 1 <<interface>> WeaponBehavior Character is the abstract class for all the other characters (King, Queen, Knight and Troll) while Weapon is an interface that all weapons implement. So all actual characters and weapons are concrete classes. To switch weapons, each character calls the setWeapon() method, which is deﬁned in the Character superclass. During a ﬁght the useWeapon() method is called on the current weapon set for a given character to inﬂict great bodily damage on another character.
