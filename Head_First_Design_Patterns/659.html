In these cases a parser/compiler generator may be more appropriate. Interpreter Uses and Drawbacks How to implement an interpreter When you need to implement a simple language, the Interpreter Pattern deﬁnes a class-based representation for its grammar along with an interpreter to interpret its sentences. To interpret the language, call the interpret() method on each expression type. This method is passed a context – which contains the input stream of the program we’re parsing – and matches the input and evaluates it.
