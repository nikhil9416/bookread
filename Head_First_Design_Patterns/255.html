Okay, we aren’t done yet; we need to work a little support into the Remote Control to handle tracking the last button pressed and the undo button press. Piece of cake! Now for the LightOffCommand. Here the undo() method just needs to call the Light’s on() method. public class LightOffCommand implements Command { Light light; public LightOffCommand(Light light) { this.light = light; } public void execute() { light.off(); } public void undo() { light.on(); } } And here, undo() turns the light back on! public class LightOnCommand implements Command { Light light; public LightOnCommand(Light light) { this.light = light; } public void execute() { light.on(); } public void undo() { light.off(); } } execute() turns the light on, so undo() simply turns the light back off. 2 Let’s start with the LightOnCommand: if the LightOnCommand’s execute() method was called, then the on() method was last called.
