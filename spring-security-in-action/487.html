We need to choose the way the resource server validates tokens issued by the authorization server. We discussed three approaches in chapters 14 and 15:  With a direct call to the authorization server  Using a blackboarding approach  With cryptographic signatures Because we already know the authorization server issues JWTs, the most comfortable choice is to rely on the cryptographic signature of the token. As you know from chap- ter 15, we need to provide the resource server the key to validate the signature.
