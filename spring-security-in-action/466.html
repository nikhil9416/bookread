Once we have a working application, we test it by call- ing the endpoint with cURL. 18.2 Configuring Keycloak as an authorization server In this section, we configure Keycloak as the authorization server for the system (fig- ure 18.3). Keycloak is an excellent open source tool designed for identity and access management. You can download Keycloak from keycloak.org. Keycloak offers the abil- ity to manage simple users locally and also provides advanced features such as user federation. You could connect it to your LDAP and Active Directory services or to dif- ferent identity providers. For example, you could use Keycloak as a high-level authen- tication layer by connecting it to one of the common OAuth 2 providers we discussed in chapter 12.
