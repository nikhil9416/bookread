For this functionality, Spring Security offers an imple- mentation called DefaultCsrfToken that we also use in our example. Default- CsrfToken implements the CsrfToken contract and creates immutable instances containing the required values: the name of the request attribute and header, and the token itself. CsrfTokenRepository is responsible for managing CSRF tokens in Spring Secu- rity. The interface CsrfTokenRepository is the contract that represents the compo- nent that manages CSRF tokens. To change the way the application manages the tokens, you need to implement the CsrfTokenRepository interface, which allows you to plug your custom implementation into the framework.
