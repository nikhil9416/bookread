The most important are  Access-Control-Allow-Origin—Specifies the foreign domains (origins) that can access resources on your domain.  Access-Control-Allow-Methods—Lets us refer only to some HTTP meth- ods in situations in which we want to allow access to a different domain, but only to specific HTTP methods. You use this if you’re going to enable exam- ple.com to call some endpoint, but only with HTTP GET, for example.  Access-Control-Allow-Headers—Adds limitations to which headers you can use in a specific request.
