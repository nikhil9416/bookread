MAPPING REACTIVE DATA One of the most common operations you’ll use on either a Flux or a Mono is to trans- form published items to some other form or type. Reactor’s types offer map() and flatMap() operations for that purpose. The map() operation creates a Flux that simply performs a transformation as prescribed by a given Function on each object it receives before republishing it.
