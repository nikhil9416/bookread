As mentioned earlier, you’ll need a MongoDB server available and listening on port 27017 to run this test. The Flapdoodle embedded MongoDB doesn’t work well with reactive repositories. If you have Docker installed on your machine, you can eas- ily start a MongoDB server exposed on port 27017 like this: $ docker run -p27017:27017 mongo Other ways to get a MongoDB setup are possible. Consult the documentation at https://www.mongodb.com/ for more details.
