This annotation is one of a handful of stereotype annotations that Spring defines, includ- ing @Controller and @Component. By annotating JdbcIngredientRepository with @Repository, you declare that it should be automatically discovered by Spring compo- nent scanning and instantiated as a bean in the Spring application context. When Spring creates the JdbcIngredientRepository bean, it injects it with Jdbc- Template. That’s because when there’s only one constructor, Spring implicitly applies autowiring of dependencies through that constructor’s parameters. If there is more than one constructor, or if you just want autowiring to be explicitly stated, then you can annotate the constructor with @Autowired as follows: @Autowired public JdbcIngredientRepository(JdbcTemplate jdbcTemplate) { this.jdbcTemplate = jdbcTemplate; } The constructor assigns JdbcTemplate to an instance variable that will be used in other methods to query and insert into the database.
