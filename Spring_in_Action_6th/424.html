INSPECTING THE ENVIRONMENT AND CONFIGURATION PROPERTIES In addition to knowing how your application beans are wired together, you might also be interested in learning what environment properties are available and what configu- ration properties were injected into the beans. When you issue a GET request to the /env endpoint, you’ll receive a rather lengthy response that includes properties from all property sources in play in the Spring appli- cation. This includes properties from environment variables, JVM system properties, application.properties and application.yml files, and even the Spring Cloud Config Server (if the application is a client of the Config Server).
