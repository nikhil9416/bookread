371 Introducing RSocket In some cases, the client may need to send data to the server but doesn’t need a response. RSocket provides the fire-and-forget model for those situations, as illustrated in figure 14.3. In the fire-and-forget model, a client sends a request to the server, but the server doesn’t send a response back. Finally, the most flexible of RSocket’s communication models is the channel model. In the channel model, the client opens a bidirectional channel with the server, and each can send data to the other at any time. Figure 14.4 illustrates the channel com- munication style. RSocket is supported on a variety of languages and platforms, including Java, Java- Script, Kotlin, .NET, Go, and C++. 1 Recent versions of Spring offer first-class support for RSocket, making it easy to create servers and clients using familiar Spring idioms. 1 This is just a short list of languages that are listed on the RSocket website, but there may be community-led implementations of RSocket for other languages.
