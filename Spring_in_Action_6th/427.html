NAVIGATING HTTP REQUEST MAPPINGS Although Spring MVC’s (and Spring WebFlux’s) programming model makes it easy to handle HTTP requests by simply annotating methods with request-mapping annota- tions, it can sometimes be challenging to get a big-picture understanding of all the kinds of HTTP requests that an application can handle and what kinds of components handle those requests. Actuator’s /mappings endpoint offers a one-stop view of every HTTP request han- dler in an application, whether it be from a Spring MVC controller or one of Actua- tor’s own endpoints.
