461 Initializing a project with Spring Tool Suite The first field on this page asks you to specify the location of the Spring Initializr service. If you’re running or using a custom instance of the Initializr, you’ll want to specify the base URL of the Initializr service here. Otherwise, you’ll be fine leaving it with the default that points to http://start.spring.io.
