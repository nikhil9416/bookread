275 Summary Summary  Spring Integration enables the definition of flows through which data can be processed as it enters or leaves an application.  Integration flows can be defined in XML, Java, or using a succinct Java DSL con- figuration style.  Message gateways and channel adapters act as entry and exit points of an inte- gration flow.  Messages can be transformed, split, aggregated, routed, and processed by ser- vice activators in the course of a flow.  Message channels connect the components of an integration flow.
