82 CHAPTER 3 Working with data As for the @Id annotation, it designates the id property as being the identity for a TacoOrder. All other properties in TacoOrder will be mapped automatically to col- umns based on their property names. For example, the deliveryName property will be automatically mapped to the column named delivery_name. But if you want to explicitly define the column name mapping, you could annotate the property with @Column like this: @Column("customer_name") @NotBlank(message="Delivery name is required") private String deliveryName; In this case, @Column is specifying that the deliveryName property will be mapped to the column whose name is customer_name.
